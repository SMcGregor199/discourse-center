# Discourse Center Demo v.0.1

Discourse Center Demo v.0.1 is a browser-based writing and research workspace for drafting long-form documents, organizing projects, managing citations, and storing reference images in one place.

Learn more about the product at [discourse.center](https://www.discourse.center/).

## What We're Building

We’re building a focused document creation environment that combines a rich text editor with lightweight research tooling. The goal is to make it easier to move from idea to draft by keeping writing, citation management, project organization, and supporting assets inside a single application.

## Running The App

Install dependencies and start the development server:

```bash
npm install
npm run dev
```

## Current Features

- Project dashboard with a first-run default project and per-project navigation
- Inline project renaming from the dashboard
- Rich text editing with headings, lists, quotes, code, bold, italic, and strike formatting
- Bubble and floating editor menus for faster formatting actions
- Auto-saving project content to local storage
- Word count and editor status feedback
- Image repository for uploading, browsing, and deleting stored images
- Notes for each stored image to capture context, reminders, and reference details
- Image insertion into documents from the shared repository
- Side-by-side image reference panel so writers can view images and notes beside the document while drafting
- Citation style selection when creating a project
- Source management for books, journal articles, websites, articles, and other references
- Generated in-text citations and works cited/reference list formatting for MLA, APA, Chicago, and Harvard
- AI research assistant modal with selectable project and image context
- Research pack insertion using the current mock scholarly search flow
